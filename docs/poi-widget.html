POI / Sehenswürdigkeiten – Widget einbinden

  1. Zugriff auf die eigene Website prüfen.
  2. Im CMS einen Baustein wie „Custom HTML“ / „Eigener Code“ / „HTML-Block“ wählen oder in einem Text-/Inhaltsfeld in den Quellcode-/HTML-Modus wechseln.
  3. Den folgenden Code einfügen (nur einmal pro Seite möglich):
<script 
  id="et4pages" 
  type="text/javascript" 
  src="https://willkommen.wangerland.de/de/wangerland/default/search/POI/view:map,half/sort:distance?i_target=et4pages">
</script>

Die Kartenansicht kann über view:map,half durch view:gallery ausgetauscht werden.).

Eine Vorbelegung auf bestimmte Kategorien (Museen, Strände, Freizeiteinrichtungen) ist möglich – Hierzu bitte direkt Kontakt aufnehmen
