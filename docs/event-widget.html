Veranstaltungen – so bindest du das Widget ein

  1. Ziel‑Element anlegen: <div id="et4pages"></div>
  2. Script hinzufügen: das ET4‑Script mit i_target=et4pages direkt darunter einfügen.
  3. Mehrere Instanzen? Eigene IDs verwenden (z. B. et4pages-1, et4pages-2).
  4. Layout beruhigen: dem Ziel‑Element eine Mindesthöhe geben (z. B. via CSS/Tailwind).
  5. DSGVO‑Hinweis: Das Widget lädt externe Inhalte; IP‑Übermittlung an den Anbieter ist technisch nötig.

Weiter unten findest du die Live‑Vorschau und zwei Copy‑Paste‑Snippets (dein Wunsch‑Snippet und die empfohlene Variante).

Direkt eingebunden

Ihr Code‑Baustein (wie gewünscht)

Diesen Snippet können Ihre Partner:innen direkt in ihre Seite einfügen.

<script 
	id="et4pages" 
	type="text/javascript" 
	src="https://willkommen.wangerland.de/de/default/search/Event/mode:next_months,3/sort:chronological?i_target=et4pages">
</script>

Hinweis: Diese Variante setzt die Ziel‑ID auf das <script> selbst. Funktioniert in vielen Fällen, kann aber bei mehrfachen Einbindungen unübersichtlich werden.

Empfohlen (sauber getrenntes Ziel‑Element)

Robust bei mehreren Instanzen und besser für Layout‑Kontrolle.

<div id="et4pages"></div>
<script defer src="https://willkommen.wangerland.de/de/wangerland/default/search/Event/Lat:53.68612670932186/Lng:8.017145762400807/all:all/mode:next_days,3/sort:chronological?i_target=et4pages"></script>
  • Quelle: willkommen.wangerland.de
  • Ziel-ID (i_target) anpassen, falls mehrfach genutzt.

Hinweis zu externen Inhalten

Auf dieser Seite erfolgt kein Tracking. Einige Widgets laden Inhalte von externen Diensten (z. B. willkommen.wangerland.de). Dabei wird technisch bedingt Ihre IP‑Adresse an den jeweiligen Anbieter übermittelt. Es werden keine Cookies zu Werbe‑ oder Analysezwecken gesetzt.
