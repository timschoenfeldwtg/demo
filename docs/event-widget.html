Veranstaltungen – Widget einbinden

  1. Zugriff auf die eigene Website prüfen.
  2. Im Content Management System (Website) ein Modulbaustein wie „Custom HTML“ / „Eigener Code“ / „HTML-Block“ nutzen oder in einem Text-/Inhaltsfeld in den Quellcode-/HTML-Modus wechseln.
  3. Den folgenden Code oder einen der unten aufgeführten Beispiele einfügen:

Veranstaltungskalender für 7 Tage

Dieser Code kann direkt in die Seite eingefügt werden.

<script 
	id="et4pages" 
	type="text/javascript" 
	src="https://willkommen.wangerland.de/de/default/search/Event/mode:next_days,7/sort:chronological?i_target=et4pages">
</script>

Hinweis: Durch Änderungen von (mode:next_days,7) kann der Zeitraum angepasst werden. Für 2 Wochen bspw. 14 statt 7 eingeben

Veranstaltungskalender mit angezeigter Karte (3-Monate)

Mit diesem Script Baustein wird der Veranstaltungskalender mit Ort auf der Karte angezeigt.

<div id="et4pages"></div>
<script defer src="https://willkommen.wangerland.de/de/wangerland/search/Event/Lat:53.68612899773418/Lng:8.017115404777904/all:all/mode:next_months,3/sort:chronological/view:map,half?i_target=et4pages"></script>
  • Die Karte kann anschließend auch wieder deaktiviert werden durch den Nutzenden
  • Mobil ist die Karte ausgeblendet.
<script 
  id="et4pages" 
  type="text/javascript" 
  src="https://willkommen.wangerland.de/de/default/search/Event/mode:next_months,3/sort:chronological?i_target=et4pages">
</script>

Hinweis: Mehrere Widgets auf einer Seite sind nicht möglich (nur einmal einbinden).

Kein Zugriff vorhanden? Den obigen Code oder diese Seite an die betreuende Agentur/den Webmaster weiterleiten und um Einbau bitten.
